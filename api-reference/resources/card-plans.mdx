---
title: "Get Card Plans"
api: "GET https://api.uncash.io/api/v1/card-plans"
description: "Get all active card plans with pricing and limits"
---

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of available card plan objects
  <Expandable title="Card Plan object">
    <ResponseField name="id" type="integer">Card plan identifier</ResponseField>
    <ResponseField name="name" type="string">Card plan name</ResponseField>
    <ResponseField name="description" type="string">Card plan description</ResponseField>
    <ResponseField name="limit" type="number">Maximum card balance limit</ResponseField>
    <ResponseField name="creation_fee" type="number">Fee charged to create this card plan</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.uncash.io/api/v1/card-plans \
  -H "Authorization: Bearer uc_your_api_token_here"
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/card-plans', {
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/card-plans"
headers = {"Authorization": "Bearer uc_your_api_token_here"}

response = requests.get(url, headers=headers)
data = response.json()
```
