---
title: "Health Check"
api: "GET https://api.uncash.io/api/v1/health"
description: "Check API health status (no authentication required)"
---

<Note>
  This endpoint does not require authentication and can be used to verify API availability.
</Note>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the API is healthy
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="status" type="string">Health status (e.g., "healthy")</ResponseField>
    <ResponseField name="environment" type="string">Deployment environment</ResponseField>
    <ResponseField name="timestamp" type="string">ISO 8601 timestamp</ResponseField>
    <ResponseField name="uptime" type="number">Server uptime in milliseconds</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.uncash.io/api/v1/health
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/health');
const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/health"
response = requests.get(url)
data = response.json()
```

</RequestExample>
