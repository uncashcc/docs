---
title: "Create Card"
api: "POST https://api.uncash.io/api/v1/cards"
description: "Create a new virtual card with automatic wallet deduction"
---

## Body Parameters

<ParamField body="bin_id" type="string" required>
  BIN identifier for the card (get from `/api/v1/bins`)
</ParamField>

<ParamField body="name" type="string" required>
  Card nickname (1-255 characters)
</ParamField>

<ParamField body="card_type" type="integer" required>
  Card type ID (get from `/api/v1/card-types`)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the card was created successfully
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Card details">
    <ResponseField name="id" type="integer">Database card ID</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp</ResponseField>
    <ResponseField name="updated_at" type="string">ISO 8601 timestamp</ResponseField>
    <ResponseField name="currency" type="string">Card currency (e.g., "USD")</ResponseField>
    <ResponseField name="EXP" type="string">Expiration date (masked)</ResponseField>
    <ResponseField name="masked_PAN" type="string">Masked card number</ResponseField>
    <ResponseField name="name" type="string">Card nickname</ResponseField>
    <ResponseField name="category" type="string">Card category</ResponseField>
    <ResponseField name="currency_short_name" type="string">Currency abbreviation</ResponseField>
    <ResponseField name="balance" type="number">Current card balance</ResponseField>
    <ResponseField name="deposit" type="number">Total deposits made</ResponseField>
    <ResponseField name="spent" type="number">Total amount spent</ResponseField>
    <ResponseField name="status" type="string">Card status (e.g., "Processing", "Active")</ResponseField>
    <ResponseField name="bin_id" type="string">Associated BIN</ResponseField>
    <ResponseField name="3ds" type="boolean">3D Secure enabled</ResponseField>
    <ResponseField name="address" type="string">Billing address</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.uncash.io/api/v1/cards \
  -H "Authorization: Bearer uc_your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "bin_id": "6705952d3ec949063b9241a0",
    "name": "My Business Card",
    "card_type": 1
  }'
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/cards', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    bin_id: "6705952d3ec949063b9241a0",
    name: "My Business Card",
    card_type: 1
  })
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards"
headers = {
    "Authorization": "Bearer uc_your_api_token_here",
    "Content-Type": "application/json"
}
data = {
    "bin_id": "6705952d3ec949063b9241a0",
    "name": "My Business Card", 
    "card_type": 1
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
```
