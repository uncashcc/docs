---
title: "Manage Auto Top-up"
api: "PUT https://api.uncash.io/api/v1/cards/{id}/auto-topup"
description: "Enable or disable automatic top-up for a virtual card"
---

## Path Parameters

<ParamField path="id" type="integer" required>
  The card ID to configure auto top-up for
</ParamField>

## Body Parameters

<ParamField body="enabled" type="boolean" required>
  Whether to enable or disable auto top-up
</ParamField>

<ParamField body="min_balance" type="number">
  Minimum balance threshold to trigger auto top-up (required when enabling, range: 1.00 - 1,000.00)
</ParamField>

<ParamField body="topup_amount" type="number">
  Amount to add when auto top-up triggers (required when enabling, range: 5.00 - 10,000.00)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the auto top-up configuration was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="message" type="string">
      Success message indicating the action performed
    </ResponseField>
    <ResponseField name="settings" type="object">
      <Expandable title="properties">
        <ResponseField name="enabled" type="boolean">
          Current auto top-up enabled status
        </ResponseField>
        <ResponseField name="min_balance" type="number">
          Configured minimum balance threshold
        </ResponseField>
        <ResponseField name="topup_amount" type="number">
          Configured top-up amount
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Validation Rules

- **enabled**: Required boolean field
- **min_balance**: Required when enabling (1.00 - 1,000.00, max 2 decimal places)
- **topup_amount**: Required when enabling (5.00 - 10,000.00, max 2 decimal places)
- **Card type**: Only FinUp cards support auto top-up

<Info>
  **How Auto Top-up Works**: When your card balance falls below the minimum threshold, the system automatically tops up the card with the specified amount from your wallet balance (plus platform fee).
</Info>
