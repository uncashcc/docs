---
title: "Withdraw from Card"
api: "POST https://api.uncash.io/api/v1/cards/{id}/withdraw"
description: "Withdraw funds from a virtual card back to your wallet"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The card ID (UUID) to withdraw from
</ParamField>

## Body Parameters

<ParamField body="amount" type="number" required>
  Amount to withdraw from the card (minimum: 1.00, maximum: 10,000.00)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the withdrawal was successful
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="card_id" type="string">Card ID (UUID) that was withdrawn from</ResponseField>
    <ResponseField name="transaction_id" type="string">Transaction ID (UUID) for this withdrawal</ResponseField>
    <ResponseField name="invoice_id" type="number">Invoice ID for this transaction</ResponseField>
    <ResponseField name="wallet_balance" type="number">New wallet balance after withdrawal</ResponseField>
    <ResponseField name="new_card_balance" type="number">New card balance after withdrawal</ResponseField>
    <ResponseField name="amount_withdrawn" type="number">Amount withdrawn</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.uncash.io/api/v1/cards/card-id/withdraw \
  -H "Authorization: Bearer uc_your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 10
  }'
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/cards/card-id/withdraw', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 10
  })
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards/card-id/withdraw"
headers = {
    "Authorization": "Bearer uc_your_api_token_here",
    "Content-Type": "application/json"
}
data = {"amount": 10}

response = requests.post(url, headers=headers, json=data)
result = response.json()
```

## Validation Rules

- **Minimum amount**: $1.00
- **Maximum amount**: $10,000.00
- **Decimal precision**: 2 decimal places maximum
- **Card balance**: Card must have sufficient balance
- **Cooldown**: 15-minute cooldown between withdrawal attempts

<Info>
  **Provider Withdrawal**: The withdrawal is first processed with the card provider, then funds are credited to your wallet. This ensures balance consistency.
</Info>
