---
title: "Withdraw from Card"
api: "POST https://api.uncash.io/api/v1/cards/{id}/withdraw"
description: "Withdraw funds from a virtual card to your wallet balance"
---

## Path Parameters

<ParamField path="id" type="integer" required>
  The card ID to withdraw from
</ParamField>

## Body Parameters

<ParamField body="amount" type="number" required>
  Amount to withdraw from the card (minimum: 1.00, maximum: 10,000.00)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the withdrawal was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="message" type="string">
      Success message with amount withdrawn
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.uncash.io/api/v1/cards/123/withdraw \
  -H "Authorization: Bearer uc_your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 50.00
  }'
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/cards/123/withdraw', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here',
    'Content-Type': 'application/json'
  },

    amount: 50.00
  })
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards/123/withdraw"
headers = {
    "Authorization": "Bearer uc_your_api_token_here",
    "Content-Type": "application/json"
}

result = response.json()
```

## Validation Rules

- **Minimum amount**: $1.00
- **Maximum amount**: $10,000.00
- **Decimal places**: Maximum 2 decimal places
- **Card balance**: Withdrawal amount cannot exceed current card balance
- **Card type**: Only FinUp cards support withdrawals

<Info>
  **Atomic Operation**: The withdrawal process is atomic - it either succeeds completely or fails completely. Your card is only debited if the wallet credit succeeds.
</Info>
