---
title: "Top Up Card"
api: "POST https://api.uncash.io/api/v1/cards/{id}/topup"
description: "Add funds to a virtual card from your wallet balance"
---

## Path Parameters

<ParamField path="id" type="integer" required>
  The card ID to top up
</ParamField>

## Body Parameters

<ParamField body="amount" type="number" required>
  Amount to add to the card (minimum: 5.00, maximum: 10,000.00)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the top-up was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="message" type="string">
      Success message with amount topped up
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.uncash.io/api/v1/cards/123/topup \
  -H "Authorization: Bearer uc_your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 100.00
  }'
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards/123/topup"
headers = {
    "Authorization": "Bearer uc_your_api_token_here",
    "Content-Type": "application/json"

```

## Validation Rules

- **Minimum amount**: $5.00
- **Maximum amount**: $10,000.00
- **Decimal places**: Maximum 2 decimal places
- **Card limits**: Top-up cannot exceed the card's maximum deposit limit
- **Wallet balance**: Must have sufficient funds in wallet

<Info>
  **Atomic Operation**: The top-up process is atomic - it either succeeds completely or fails completely. Your wallet is only charged if the card top-up succeeds.
</Info>
