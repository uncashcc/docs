---
title: "Top Up Card"
api: "POST https://api.uncash.io/api/v1/cards/{id}/topup"
description: "Add funds to a virtual card from your wallet balance"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The card ID (UUID) to top up
</ParamField>

## Body Parameters

<ParamField body="amount" type="number" required>
  Amount to add to the card (minimum: 5.00, maximum: 10,000.00)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the top-up was successful
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="card_id" type="string">Card ID (UUID) that was topped up</ResponseField>
    <ResponseField name="transaction_id" type="string">Transaction ID (UUID) for this topup</ResponseField>
    <ResponseField name="invoice_id" type="number">Invoice ID for this transaction</ResponseField>
    <ResponseField name="wallet_balance" type="number">New wallet balance after topup</ResponseField>
    <ResponseField name="total_charged" type="number">Total amount charged from wallet (amount + platform fee)</ResponseField>
    <ResponseField name="platform_fee_amount" type="number">Platform fee charged for this topup</ResponseField>
    <ResponseField name="new_card_balance" type="number">New card balance after topup</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.uncash.io/api/v1/cards/card-id/topup \
  -H "Authorization: Bearer uc_your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 5
  }'
```


## Validation Rules

- **Minimum amount**: $5.00
- **Maximum amount**: $10,000.00
- **Decimal precision**: 2 decimal places maximum
- **Card limit**: Topup amount + current deposit cannot exceed card limit
- **Wallet balance**: Must have sufficient balance (topup amount + platform fee)
- **Cooldown**: 2-minute cooldown between topup attempts on the same card

<Info>
  **Platform Fee**: A platform fee is charged based on the card type. The fee is automatically calculated and included in the `total_charged` amount deducted from your wallet.
</Info>
