---
title: "Request Card OTP"
api: "POST https://api.uncash.io/api/v1/cards/{id}/otps/request"
description: "Request to retrieve OTP from email for a specific card"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The card ID to request OTP for
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Response data">
    <ResponseField name="message" type="string">Success message</ResponseField>
    <ResponseField name="cardInfo" type="object">
      Basic card information
      <Expandable title="Card Info">
        <ResponseField name="id" type="string">Card ID</ResponseField>
        <ResponseField name="providerCardId" type="string">Provider card ID</ResponseField>
        <ResponseField name="name" type="string">Card name/nickname</ResponseField>
        <ResponseField name="otp_code" type="string">OTP code</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.uncash.io/api/v1/cards/abc123/otps/request \
  -H "Authorization: Bearer uc_your_api_token_here"
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/cards/abc123/otps/request', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards/abc123/otps/request"
headers = {
    "Authorization": "Bearer uc_your_api_token_here"
}

response = requests.post(url, headers=headers)
data = response.json()
```
