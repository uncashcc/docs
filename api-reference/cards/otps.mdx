---
title: "Get Card OTPs"
api: "GET https://api.uncash.io/api/v1/cards/{id}/otps"
description: "Retrieve the 5 most recent OTPs (One-Time Passwords) for a specific card"
---

## Path Parameters

<ParamField path="id" type="integer" required>
  The card ID to retrieve OTPs for
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="OTPs data">
    <ResponseField name="otps" type="array">
      Array of the 5 most recent OTPs, sorted by creation date (newest first)
      <Expandable title="OTP Object">
        <ResponseField name="id" type="string">Unique OTP identifier</ResponseField>
        <ResponseField name="code" type="string">6-digit OTP code</ResponseField>
        <ResponseField name="amount" type="number">Transaction amount for this OTP</ResponseField>
        <ResponseField name="currency" type="string">Transaction currency</ResponseField>
        <ResponseField name="masked_pan" type="string">Masked card number</ResponseField>
        <ResponseField name="card_name" type="string">Card name/nickname</ResponseField>
        <ResponseField name="created_at" type="string">ISO 8601 timestamp when OTP was created</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="cardInfo" type="object">
      Basic card information
      <Expandable title="Card Info">
        <ResponseField name="id" type="integer">Card ID</ResponseField>
        <ResponseField name="nickname" type="string">Card nickname</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.uncash.io/api/v1/cards/123/otps \
  -H "Authorization: Bearer uc_your_api_token_here"
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/cards/123/otps', {
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards/123/otps"
headers = {"Authorization": "Bearer uc_your_api_token_here"}

response = requests.get(url, headers=headers)
data = response.json()
```
