---
title: "Close Card"
api: "PATCH https://api.uncash.io/api/v1/cards/{id}/close"
description: "Close a virtual card and return any remaining balance to wallet"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The card ID to close
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the card was closed successfully
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="card_id" type="string">Card ID that was closed</ResponseField>
    <ResponseField name="balance_withdrawn" type="number">Amount returned to wallet from card balance</ResponseField>
    <ResponseField name="transaction_id" type="string">Transaction ID for the balance withdrawal (if any)</ResponseField>
    <ResponseField name="new_wallet_balance" type="number">New wallet balance after balance return</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X PATCH https://api.uncash.io/api/v1/cards/123/close \
  -H "Authorization: Bearer uc_your_api_token_here"
```

```javascript JavaScript
const response = await fetch('https://api.uncash.io/api/v1/cards/123/close', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer uc_your_api_token_here'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.uncash.io/api/v1/cards/123/close"
headers = {"Authorization": "Bearer uc_your_api_token_here"}

response = requests.patch(url, headers=headers)
data = response.json()
```

</RequestExample>

<Info>
  **Balance Refund**: Any remaining balance on the card is automatically returned to your wallet when the card is closed. This operation cannot be undone.
</Info>

<Warning>
  **Rate Limiting**: Must wait 15 minutes between withdrawals. This applies to card closures that trigger automatic balance withdrawals.
</Warning>
