## Welcome to Uncash API

The Uncash API is a production-ready Cloudflare Workers API that provides comprehensive virtual card management capabilities. Built with security, performance, and developer experience in mind.

<Note>
  This API is designed for financial applications and implements enterprise-grade security measures including database locking, input validation, and authentication.
</Note>

## Key Features

- **Virtual Card Management**: Create, fund, and manage virtual payment cards
- **Wallet System**: Built-in wallet with transaction tracking
- **FinUP Integration**: Seamless integration with FinUP card provider
- **Security First**: API token authentication, rate limiting, and input validation
- **Cloudflare Workers**: Global edge deployment for low latency
- **Transaction Safety**: Database-level locking prevents race conditions

## Base URL

```bash
https://api.uncash.io
```

## Authentication

All API requests require authentication using API tokens:

```bash
Authorization: Bearer uc_your_api_token_here
```

<Warning>
  Keep your API tokens secure and never expose them in client-side code. All financial operations require valid authentication.
</Warning>
