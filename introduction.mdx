## Uncash API

Production-ready API for virtual card management with wallet integration.

## Base URL

```bash
https://api.uncash.io
```

## Authentication

```bash
Authorization: Bearer uc_your_api_token_here
```

Get your API token from [dashboard/developer](https://uncash.io/dashboard/developer)
